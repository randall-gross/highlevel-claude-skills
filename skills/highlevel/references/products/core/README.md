# Products — Core

Core CRUD operations for products including list, get, create, update, delete, and bulk operations.

## Endpoints

| File | Method | Path | Description |
|------|--------|------|-------------|
| `list.md` | GET | `/products/` | List products with search and pagination |
| `get.md` | GET | `/products/{productId}` | Get a single product by ID |
| `create.md` | POST | `/products/` | Create a new product |
| `update.md` | PUT | `/products/{productId}` | Update an existing product |
| `delete.md` | DELETE | `/products/{productId}` | Delete a product |
| `bulk-update.md` | POST | `/products/bulk-update` | Bulk update products |
| `bulk-edit.md` | POST | `/products/bulk-update/edit` | Bulk edit products and prices |
