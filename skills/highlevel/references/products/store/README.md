# Products — Store

Store-level product management. Control which products appear in a store, set display priorities, and retrieve product stats.

## Endpoints

| File | Method | Path | Description |
|------|--------|------|-------------|
| `include-exclude.md` | POST | `/products/store/{storeId}` | Include or exclude a product in a store |
| `priorities.md` | POST | `/products/store/{storeId}/priority` | Update product display priorities |
| `stats.md` | GET | `/products/store/{storeId}/stats` | Get store product stats |
