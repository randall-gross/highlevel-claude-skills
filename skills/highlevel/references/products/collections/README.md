# Products — Collections

CRUD operations for product collections. Collections group products for storefront organization and display.

## Endpoints

| File | Method | Path | Description |
|------|--------|------|-------------|
| `list.md` | GET | `/products/collections` | List all collections for a location |
| `get.md` | GET | `/products/collections/{collectionId}` | Get a single collection by ID |
| `create.md` | POST | `/products/collections` | Create a new collection |
| `update.md` | PUT | `/products/collections/{collectionId}` | Update an existing collection |
| `delete.md` | DELETE | `/products/collections/{collectionId}` | Delete a collection |
