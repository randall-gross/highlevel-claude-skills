# Custom Objects — Schemas

Schema operations for custom objects. Schemas define the structure (fields, labels) of a custom object type.

## Endpoints

| File | Method | Path | Description |
|------|--------|------|-------------|
| `list.md` | GET | `/objects/` | List all custom object schemas for a location |
| `get.md` | GET | `/objects/{key}` | Get a single schema by key |
| `create.md` | POST | `/objects/` | Create a new custom object schema |
| `update.md` | PUT | `/objects/{key}` | Update an existing schema |
