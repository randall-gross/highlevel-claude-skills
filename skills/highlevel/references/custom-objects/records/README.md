# Custom Objects — Records

Record operations for custom objects. Records are individual data entries that conform to a schema's field definitions.

## Endpoints

| File | Method | Path | Description |
|------|--------|------|-------------|
| `get.md` | GET | `/objects/{schemaKey}/records/{id}` | Get a single record by ID |
| `create.md` | POST | `/objects/{schemaKey}/records` | Create a new record |
| `update.md` | PUT | `/objects/{schemaKey}/records/{id}` | Update an existing record |
| `delete.md` | DELETE | `/objects/{schemaKey}/records/{id}` | Delete a record |
| `search.md` | POST | `/objects/{schemaKey}/records/search` | Search records with filters |
