# Opportunities — Core

Core CRUD operations for opportunities including get, create, update, upsert, delete, search, and status updates.

## Endpoints

| File | Method | Path | Description |
|------|--------|------|-------------|
| `get.md` | GET | `/opportunities/{id}` | Get a single opportunity by ID |
| `create.md` | POST | `/opportunities/` | Create a new opportunity |
| `update.md` | PUT | `/opportunities/{id}` | Update an existing opportunity |
| `upsert.md` | POST | `/opportunities/upsert` | Create or update by contact+pipeline match |
| `delete.md` | DELETE | `/opportunities/{id}` | Delete an opportunity |
| `search.md` | GET | `/opportunities/search` | Search opportunities with filters |
| `update-status.md` | PUT | `/opportunities/{id}/status` | Update opportunity status only |
