# Delete Schedule

- **Method:** DELETE
- **Path:** `/invoices/schedule/{scheduleId}`
- **Scopes:** `invoices/schedule.write`
- **Rate Limit:** Standard (100/10s)

## Parameters

### Path Parameters
| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| scheduleId | string | Yes | Schedule ID |

### Query Parameters
| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| altId | string | Yes | Location ID |
| altType | string | Yes | Must be `location` |

## Request Example
```bash
curl -X DELETE "https://services.leadconnectorhq.com/invoices/schedule/sched_abc123?altId=ve9EPM428h8vShlRW1KT&altType=location" \
  -H "Authorization: Bearer TOKEN" \
  -H "Content-Type: application/json" \
  -H "Version: 2021-07-28"
```

## Response Example
```json
{
  "success": true
}
```

## Error Responses
| Status | Meaning |
|--------|---------|
| 400 | Missing altId or altType |
| 401 | Unauthorized — invalid or expired token |
| 404 | Schedule not found |
| 422 | Cannot delete active schedule (cancel first) |

## Common Use Cases
Remove a draft or cancelled schedule that is no longer needed.

## Gotchas
- Active schedules must be cancelled before they can be deleted.
- Deletion does not affect invoices already generated by the schedule.
