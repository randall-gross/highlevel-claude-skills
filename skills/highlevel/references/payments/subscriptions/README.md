# Subscriptions Subdomain

Query recurring subscriptions.

## Scopes

- `payments/subscriptions.readonly` — List and get subscriptions

## Endpoints

| Method | Path | File | Description |
|--------|------|------|-------------|
| GET | `/payments/subscriptions` | [list.md](list.md) | List subscriptions |
| GET | `/payments/subscriptions/{subscriptionId}` | [get.md](get.md) | Get subscription by ID |
