# Transactions Subdomain

Query payment transactions.

## Scopes

- `payments/transactions.readonly` — List and get transactions

## Endpoints

| Method | Path | File | Description |
|--------|------|------|-------------|
| GET | `/payments/transactions` | [list.md](list.md) | List transactions |
| GET | `/payments/transactions/{transactionId}` | [get.md](get.md) | Get transaction by ID |
