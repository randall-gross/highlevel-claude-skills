# Integrations Subdomain

Manage white-label payment provider integrations.

## Scopes

- `payments/integration.readonly` — List white-label providers
- `payments/integration.write` — Create white-label providers

## Endpoints

| Method | Path | File | Description |
|--------|------|------|-------------|
| GET | `/payments/integrations/provider/whitelabel` | [list-whitelabel.md](list-whitelabel.md) | List white-label providers |
| POST | `/payments/integrations/provider/whitelabel` | [create-whitelabel.md](create-whitelabel.md) | Create white-label provider |
