# Courses API

Import courses and course content into HighLevel locations.

## Required Scopes
- `courses.readonly`
- `courses.write`

## Endpoints

| File | Method | Path | Description |
|------|--------|------|-------------|
| [import.md](core/import.md) | POST | `/courses/courses-exporter/public/import` | Import a course |
