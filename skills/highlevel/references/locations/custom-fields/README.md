# Locations — Custom Fields

Manage custom field definitions for a location. Custom fields extend the data model for contacts, opportunities, and other objects.

## Endpoints

| Method | Path | File | Description |
|--------|------|------|-------------|
| GET | `/locations/{locationId}/customFields` | [list.md](list.md) | List custom fields |
| GET | `/locations/{locationId}/customFields/{id}` | [get.md](get.md) | Get custom field |
| POST | `/locations/{locationId}/customFields` | [create.md](create.md) | Create custom field |
| PUT | `/locations/{locationId}/customFields/{id}` | [update.md](update.md) | Update custom field |
| DELETE | `/locations/{locationId}/customFields/{id}` | [delete.md](delete.md) | Delete custom field |
| POST | `/locations/{locationId}/customFields/upload` | [upload.md](upload.md) | Upload file to custom field |
