# Locations — Tags

Manage tags for a location. Tags are used to categorize and segment contacts and opportunities.

## Endpoints

| Method | Path | File | Description |
|--------|------|------|-------------|
| GET | `/locations/{locationId}/tags` | [list.md](list.md) | List all tags |
| GET | `/locations/{locationId}/tags/{tagId}` | [get.md](get.md) | Get tag |
| POST | `/locations/{locationId}/tags` | [create.md](create.md) | Create tag |
| PUT | `/locations/{locationId}/tags/{tagId}` | [update.md](update.md) | Update tag |
| DELETE | `/locations/{locationId}/tags/{tagId}` | [delete.md](delete.md) | Delete tag |
