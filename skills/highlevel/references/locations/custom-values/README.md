# Locations — Custom Values

Manage reusable value snippets for a location. Custom values store frequently-used text like company name, address, phone number, etc. that can be inserted into templates and messages.

## Endpoints

| Method | Path | File | Description |
|--------|------|------|-------------|
| GET | `/locations/{locationId}/customValues` | [list.md](list.md) | List custom values |
| GET | `/locations/{locationId}/customValues/{id}` | [get.md](get.md) | Get custom value |
| POST | `/locations/{locationId}/customValues` | [create.md](create.md) | Create custom value |
| PUT | `/locations/{locationId}/customValues/{id}` | [update.md](update.md) | Update custom value |
| DELETE | `/locations/{locationId}/customValues/{id}` | [delete.md](delete.md) | Delete custom value |
