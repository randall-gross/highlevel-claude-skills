# Locations — Templates

Manage email and SMS templates for a location.

## Endpoints

| Method | Path | File | Description |
|--------|------|------|-------------|
| GET | `/locations/{locationId}/templates` | [list.md](list.md) | List email/SMS templates |
| DELETE | `/locations/{locationId}/templates/{id}` | [delete.md](delete.md) | Delete template |
