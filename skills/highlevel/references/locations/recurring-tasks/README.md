# Locations — Recurring Tasks

Manage recurring task definitions for a location. Recurring tasks are templates that automatically generate task instances on a schedule.

## Endpoints

| Method | Path | File | Description |
|--------|------|------|-------------|
| GET | `/locations/{locationId}/recurring-tasks/{id}` | [get.md](get.md) | Get recurring task |
| POST | `/locations/{locationId}/recurring-tasks` | [create.md](create.md) | Create recurring task |
| PUT | `/locations/{locationId}/recurring-tasks/{id}` | [update.md](update.md) | Update recurring task |
| DELETE | `/locations/{locationId}/recurring-tasks/{id}` | [delete.md](delete.md) | Delete recurring task |
