# Calendars — Groups

Manage calendar groups for organizing calendars together.

## Endpoint Index

| File | Method | Path | Description |
|------|--------|------|-------------|
| [list.md](list.md) | GET | `/calendars/groups` | List calendar groups |
| [create.md](create.md) | POST | `/calendars/groups` | Create group |
| [update.md](update.md) | PUT | `/calendars/groups/{groupId}` | Update group |
| [delete.md](delete.md) | DELETE | `/calendars/groups/{groupId}` | Delete group |
| [status.md](status.md) | PUT | `/calendars/groups/{groupId}/status` | Enable/disable group |
| [validate-slug.md](validate-slug.md) | POST | `/calendars/groups/validate-slug` | Check slug availability |
