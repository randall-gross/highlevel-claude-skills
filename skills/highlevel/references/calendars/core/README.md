# Calendars — Core

CRUD operations for calendars and free slot availability.

## Endpoint Index

| File | Method | Path | Description |
|------|--------|------|-------------|
| [list.md](list.md) | GET | `/calendars/` | List all calendars |
| [get.md](get.md) | GET | `/calendars/{calendarId}` | Get single calendar |
| [create.md](create.md) | POST | `/calendars/` | Create calendar |
| [update.md](update.md) | PUT | `/calendars/{calendarId}` | Update calendar |
| [delete.md](delete.md) | DELETE | `/calendars/{calendarId}` | Delete calendar |
| [free-slots.md](free-slots.md) | GET | `/calendars/{calendarId}/free-slots` | Get available booking slots |
