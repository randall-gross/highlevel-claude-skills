# Custom Menus API

Create and manage custom menu links within HighLevel locations.

## Required Scopes
- `custom-menu-link.readonly`
- `custom-menu-link.write`

## Endpoints

| File | Method | Path | Description |
|------|--------|------|-------------|
| [list.md](core/list.md) | GET | `/custom-menus/` | List all custom menus |
| [get.md](core/get.md) | GET | `/custom-menus/{customMenuId}` | Get a custom menu |
| [create.md](core/create.md) | POST | `/custom-menus/` | Create a custom menu |
| [update.md](core/update.md) | PUT | `/custom-menus/{customMenuId}` | Update a custom menu |
| [delete.md](core/delete.md) | DELETE | `/custom-menus/{customMenuId}` | Delete a custom menu |

> This domain has placeholder endpoint files. Full documentation with request/response examples is planned for a future update.
